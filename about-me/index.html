  • Text language detection using n-grams:
    In this project, the language of an input text was recognized using n-grams. In this way, a series of pre-defined words were available in several languages, including Persian and English. The words of the input text were extracted and hashed in order to find out the number of repetitions of each word. 50 words with the highest number of repetitions were compared with pre-defined words in each language, and the number of words that were the same in both were obtained. The language corresponding to the maximum of these numbers was considered as the language of the input text.

Then, semantic actions were added to it to obtain a feature grammar. Its lexical and syntactic analyzers were created with the help of the Coco/R compiler generator. Finally, a C# main program was created that contained the macro definitions for these two.
